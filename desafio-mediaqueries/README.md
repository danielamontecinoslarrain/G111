## Descripción

Este desafío consistió en desarrollar una página web para la visualización de productos, replicando el diseño y la estructura proporcionados por el docente como referencia. El objetivo fue reproducir fielmente los layouts indicados, respetando la distribución de los elementos y adaptando la propuesta a una temática libre.

Para este proyecto se eligió la temática de una tienda de mascotas, utilizando productos, imágenes y colores personalizados, manteniendo la estructura y comportamiento solicitados en el ejemplo entregado.

El desarrollo contempló la implementación de dos layouts responsivos mediante CSS Grid, Flexbox y Media Queries, permitiendo que la página se adapte correctamente a distintos tamaños de pantalla.
