# Pet Shop | Prueba M2 - Animaciones con CSS

Proyecto web responsivo de temática libre desarrollado para la prueba **Animaciones con CSS** de Desafío Latam.

## Objetivo

Construir una página web con menú, galería principal, efectos visuales, footer y diseño adaptable a distintos tamaños de pantalla.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- Transiciones y transformaciones CSS

## Características principales

- Menú con logo y enlaces a secciones.
- En dispositivos pequeños, el menú queda en la parte superior.
- En pantallas desde 576px, el menú se adapta como menú lateral.
- Galería de productos responsiva.
- Efecto visual en todas las imágenes de la galería: blanco y negro a color, escala y rotación suave al hacer hover.
- Footer con información de contacto y enlaces a redes sociales.
- Diseño coherente con paleta de colores, cards, botones y estilos relacionados al tema Pet Shop.

## Breakpoints

- Mobile first: una columna.
- Desde 576px: menú lateral y galería en dos columnas.
- Desde 992px: galería en cuatro columnas.
